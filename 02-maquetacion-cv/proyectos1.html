Programación gráfica

Emulación de fuerzas sobre un puente

1. Introducción

El objetivo de este proyecto fue desarrollar una interfaz en C++ Builder 12 utilizando FMX que permita:

  • Ingresar tres fuerzas con su respectiva magnitud y dirección (ángulo en grados).
  • Calcular la fuerza resultante en una junta crítica de un puente.
  • Determinar si el sistema se encuentra en equilibrio.
  • Graficar las fuerzas y la fuerza resultante en un plano cartesiano.

Esto permite validar si la estructura mantiene su estabilidad (resultante ≈ 0) o si existe riesgo de colapso.

2. Modelado Matemático

Cuando varias fuerzas actúan sobre un punto, se pueden descomponer en sus componentes X e Y usando fórmulas trigonométricas:

  • Fx = F * cos(θ)
  • Fy = F * sin(θ)

La suma de todas las componentes da la fuerza neta en cada dirección:

  • Fx_total = Fx1 + Fx2 + Fx3
  • Fy_total = Fy1 + Fy2 + Fy3

La magnitud de la fuerza resultante se calcula usando el teorema de Pitágoras:

  • Fr = sqrt(Fx_total^2 + Fy_total^2)

Si Fr es cercano a cero (por debajo de un umbral definido, en este caso 1.0 N), el sistema se considera en equilibrio.

3. Implementación en C++ Builder

3.1 Entradas de usuario:

  • Magnitud de cada fuerza (N).
  • Ángulo de cada fuerza (grados).

3.2 Procesamiento de datos:

  • Conversión de grados a radianes.
  • Cálculo de componentes X y Y para cada fuerza.
  • Suma de componentes para obtener la fuerza total.
  • Evaluación de la magnitud de la fuerza resultante.
  • Determinación de equilibrio del sistema.

3.3 Visualización de resultados:

  • Resultados numéricos mostrados en un Memo (Memo1).
  • Gráfica de las fuerzas mediante un Image (ImageCanvas).

Las fuerzas se dibujan como líneas de colores diferentes:

  • F1: Rojo
  • F2: Verde
  • F3: Azul
  • Fuerza resultante: Negro

4. Código Base

El archivo principal (Unit1.cpp) contiene:

  • Manejo de errores para entradas inválidas.
  • Cálculo vectorial de las fuerzas.
  • Dibujo automático de los vectores en pantalla.
  • Indicador visual de la estabilidad de la junta.

Se emplearon funciones estándar de math.h (cos, sin, sqrt) y se utilizó TAlphaColors para asignar colores a los vectores.

5. Resultados

La aplicación permite:

  • Introducir valores personalizados de magnitud y ángulo para cada fuerza.
  • Obtener el análisis de equilibrio en segundos.
  • Visualizar gráficamente la dirección y magnitud de cada fuerza.
  • Detectar fácilmente si el sistema se encuentra estable o inestable.

Esto proporciona una herramienta útil para el análisis preliminar de juntas estructurales en proyectos de ingeniería civil.

6. Conclusiones

  • La correcta descomposición y suma de fuerzas es fundamental para evaluar la estabilidad de estructuras.
  • Un sistema gráfico ayuda a interpretar de forma visual la situación de cargas.
  • C++ Builder, utilizando FMX, ofrece una plataforma eficiente para integrar cálculos matemáticos y representación visual en un solo entorno.

El proyecto cumple con los objetivos planteados y sirve como base para desarrollos más avanzados en análisis estructural en 2D.
